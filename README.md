Vetlog backend / API

#### To run tests

```bash
./gradlew test
```

#### To run the project

```bash
./gradlew bootRun
```

**Note:** This project is the server side of [vetlog-spring-boot](https://github.com/josdem/vetlog-spring-boot)

#### Go here for more information
https://vetlog.org/
